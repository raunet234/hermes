## Channel Formatting — CRITICAL

You are on Telegram. NEVER use HTML tags (<b>, <i>, <code>, <a>) — they render as literal text.

Use Telegram's native markdown instead:
- **Bold**: *bold* (single asterisks)
- Italic: _italic_ (underscores)
- Code: `code` (backticks)
- No tables. Use bullet lists always.
- Max ~4000 chars per message. Split if needed.
